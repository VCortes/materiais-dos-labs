TUTORIAL VIRTUALAB

CONOCIENDO LOS TIPOS DE CIMENTACIONES

Visualiza las cimentaciones haciendo clic con el botón izquierdo del mouse en la cámara "Subterráneo" en el menú "Visualización" o usando el atajo "Alt + 2".

Observa con atención los 5 tipos de cimentaciones.

Conoce las características de cada tipo de cimentación pasando el mouse sobre cada una de ellas.

Identifica cada una de las cimentaciones haciendo clic izquierdo en cada una de ellas.

Elije el tipo correspondiente a cada cimentación, entre zapata, pozo de cimentación, pilote prefabricado, pilote Franki, o hélice continua, pulsando con el botón izquierdo del mouse sobre una de las alternativas.
