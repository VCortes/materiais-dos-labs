DESCRIPCIÓN DEL LABORATORIO


MATERIALES NECESARIOS

  • Plantilla de prueba;

  • Placa base;

  • Agua;

  • Hormigón fresco;

  • Concha;

  • Varilla de compactación;

  • Espátula;

  • Regla graduada.


PROCEDIMIENTOS

Según la norma, el tiempo total para realizar este procedimiento es de 150 segundos. Si no puede completar este tiempo, reinicie el experimento.


HUMEDECER EL MOLDE DE PRUEBA Y LA PLACA BASE

Humedezca el molde de prueba y la placa base. El molde debe colocarse firmemente sobre la placa, que a su vez debe colocarse sobre una superficie plana, rígida y horizontal.


RELLENAR LA PLANTILLA DE PRUEBA

Rellene el molde con la muestra de hormigón fresco, formando tres capas de igual altura, compactando cada una de las capas con 25 pasadas de la varilla de compactación.


FINALIZACIÓN DEL LLENADO DEL MOLDE

Después de la tercera compactación, añada la cantidad de hormigón necesaria para llenar completamente el molde de prueba y utilice una espátula para raspar la superficie del molde.


EXTRACCIÓN DEL MOLDE DE PRUEBA

Retire el molde de prueba en menos de 10 segundos, asegurándose de que el hormigón no se someta a movimientos de torsión lateral.


MEDICIÓN DE LA REDUCCIÓN DEL HORMIGÓN

Coloque el molde de ensayo junto al cono de hormigón y, con la varilla de compactación y la regla graduada, mida la reducción del hormigón en milímetros, acercándolo a los 5 mm más próximos.

HUMEDECER EL MOLDE DE PRUEBA Y LA PLACA BASE

La prueba de hundimiento es un procedimiento muy utilizado en la construcción civil. Esta prueba consiste en determinar la consistencia del hormigón abatiendo el tronco cónico. El procedimiento presentado en este laboratorio sigue las pautas proporcionadas por la NBR NM 67.

Utilice la piseta para humedecer el molde de ensayo y la placa base.



Importante: según la norma, el tiempo total para realizar este procedimiento es de 150 segundos. Si no puede completar esta vez, una de las opciones en el modelo de ensayo es eliminar el contenido, lo que permite volver a realizar el experimento.

Consejo: Para optimizar el tiempo durante el experimento, lea todo el procedimiento presentado en este paso, identificando los comandos para cada acción en el laboratorio virtual.


RELLENAR LA PLANTILLA DE PRUEBA

El molde debe rellenarse con tres capas, cada una de aproximadamente un tercio de la altura del molde de prueba. Cada carcasa utilizada para insertar el hormigón No, el molde llena aproximadamente una novena parte del molde, por lo que cada capa de hormigón se obtiene después de usar la carcasa tres veces para insertar el hormigón.

La varilla de compactación debe usarse cada vez que se complete una capa, distribuyendo 25 golpes de manera uniforme sobre la sección de cada capa, espesando así el hormigón en el molde.

Utilice la carcasa para añadir hormigón al molde de ensayo, llenando así la primera capa.



Utilice la barra de compactación para compactar el hormigón en el molde. Repite los procedimientos de añadir el hormigón y compactarlo dos veces más, hasta compactar la tercera capa de hormigón.



FINALIZACIÓN DEL LLENADO DEL MOLDE

Una vez compactadas las tres capas, se debe insertar una última capa de hormigón fresco para garantizar que el molde se haya llenado por completo. Se debe usar una espátula para raspar la superficie de concreto con el borde del molde.

Use la carcasa para agregar concreto al molde sin prueba y mueva el tronco cónico al piso, luego use la espátula para nivelar la capa superior del molde.



EXTRACCIÓN DEL MOLDE DE PRUEBA

Para llevar a cabo la medición, se debe levantar el molde en dirección vertical y con una velocidad constante, sin que el hormigón esté sometido a movimientos de torsión lateral.

Extraiga el molde de prueba.



MEDICIÓN DE LA REDUCCIÓN DEL HORMIGÓN

La varilla de compactación debe utilizarse para facilitar la medición de la reducción. Al colocar la varilla apoyada en el molde de ensayo, es posible obtener un punto de referencia adecuado para utilizar la regla para medir (en milímetros) la reducción del hormigón.

Coloque la varilla de compactación para facilitar la medición.



Mida el asentamiento con la regla.



Consejo: Utilice la ventana Ver para buscar una opción que facilite la medición de la liquidación. Recuerde utilizar las funciones de mover y acercar/alejar la cámara, presentadas en el paso 1.
